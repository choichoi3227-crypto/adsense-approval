Nginx에 Let's Encrypt SSL 인증서 설치하기

HTTPS는 더 이상 선택이 아닙니다. Let's Encrypt 덕분에 무료로, 10분 안에 설정할 수 있습니다.

Certbot 설치

apt install certbot python3-certbot-nginx

Certbot은 Let's Encrypt 인증서 발급과 Nginx 설정 변경을 자동으로 처리해주는 공식 클라이언트입니다.

인증서 발급

certbot --nginx -d example.com -d www.example.com

이메일 주소와 약관 동의 후 도메인 소유 확인이 자동으로 이루어집니다. 성공하면 Nginx 설정에 SSL 관련 지시어가 자동으로 추가됩니다.

자동 갱신 설정

Let's Encrypt 인증서 유효 기간은 90일입니다. Certbot 설치 시 cron 또는 systemd 타이머가 자동으로 등록되어 만료 30일 전에 자동 갱신이 시도됩니다.

# 갱신 테스트 (실제 발급 없이)
certbot renew --dry-run

HTTPS 강제 리다이렉트

Certbot이 자동으로 HTTP → HTTPS 리다이렉트 설정을 추가하지만, 수동으로 추가해야 할 경우 Nginx server 블록에 다음을 추가합니다.

server {
    listen 80;
    server_name example.com;
    return 301 https://$host$request_uri;
}

인증서 상태 확인

certbot certificates 명령으로 현재 발급된 인증서 목록과 만료일을 확인할 수 있습니다. 정기적으로 확인해 자동 갱신이 정상 동작하는지 점검하는 것이 좋습니다.
